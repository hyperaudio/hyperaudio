Typefaces.

Hyperaudio body text is set in Merriweather Sans, whereas Exo is used for headings. Both fonts are served by Google WebFonts.

$primary-font      : 'Merriweather Sans', sans-serif;
$secondary-font    : 'Exo', sans-serif;

Font families with their fallbacks are defined in _config.scss. Please use those variables instead of raw values.

Bad:

font-family: 'Merriweather Sans', sans-serif;

Good: 

font-family: $primary-font

Type variables.

A set of font-size and line-height variables have been provided in order to keep those values consistent throughout the UI. Avoid specifying values of those properties in raw numbers, use following instead:

// Font Sizes
$font-h            : 4rem;
$font-xl           : 2.9rem;
$font-l            : 2.5rem;
$font-m            : 1.4rem;
$font-s            : 1.3rem;
$font-xs           : 1.1rem;

// Line Heights
$lheight-xl        : 2.4em;
$lheight-l         : 1.8em;
$lheight-m         : 1.6em;
$lheight-s         : 1.4em;
$lheight-xs        : 1.2em;

Bad:

font-size: 2.9rem;
line-height: 1.4em;

Good:

font-size: $font-xl;
line-height: $lheight-s;

Headings. .heading-[modifier]

Headings depend on classes. This way a span can easily look like a heading if only it has the right class applied.

That said, h1-h5 use SASS @extends to suck on heading classes so placing a raw h1 anywhere will have the same effect as placing a h1.heading-xl. Heading class index is the following:

.heading-xl
.heading-l
.heading-m
.heading-s
.heading-xs

Heading XL

This is a heading

Heading L

This is a heading

Heading M

This is a heading

Heading S

This is a heading

Heading XS

This is a heading

Tables. .table-[modifier]

Tables come in one variation. The only personalizations one can apply to them is setting the width of each cell by adding modifier class as listed:

  • .span1 that stands for 10%
  • .span2 that stands for 20%
  • .span3 that stands for 30%
  • .span4 that stands for 40%
  • .span5 that stands for 50%

You’re not bound to always use complete table structure—feel free to mark it up as you wish by dropping caption, thead or tfoot if necessary.
