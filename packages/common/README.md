# remixer

## Install

```bash
npm install --save @hyperaudio/common
```

## Usage

```jsx
import React, { Component } from 'react';

import MyComponent from '@hyperaudio/common';
import '@hyperaudio/common/dist/index.css';

class Example extends Component {
  render() {
    return <MyComponent />;
  }
}
```
